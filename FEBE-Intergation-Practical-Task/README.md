## Installation

```bash
# Clone the repository
git clone https://github.com/Russell-Fulwood/codingTasks.git

# Navigate to the project directory
cd codingTasks/FEBE-Intergation-Practical-Task

# backend
cd backend

# Install dependencies
npm install

# frontend
cd ../frontend

# Install dependencies
npm install
```

## Usage

```bash
# Run the backend 
npm start

# next run the frontend
npm run dev
```
